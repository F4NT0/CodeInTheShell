# TPUT COLORS

* Stop Color: `tput sgr0`

### Foreground Colors

* We use the `setaf` command with tput

Command|Color|
|---|---|
`tput setaf 1`|Red
`tput setaf 2`|Green
`tput setaf 3`|Orange
`tput setaf 4`|Blue
`tput setaf 5`|Magenta
`tput setaf 6`|Yellow
`tput setaf 7`|White
`tput setaf 8`|Black
